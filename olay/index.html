Olay

Livestream overlay stuff for use as Browser-Source in OBS Studio.

  1. Test and setup the overlays with the forms here. If you do not enter any values, defaults will be used.
  2. When you're happy with the output, copy & paste the URL into OBS' Browser-Source.
  3. Remove all code from the CSS field of the Browser-Source.
  4. Copy & paste the default CSS (linked for each module below) into the CSS field of the Browser-Source and start adjusting it to your liking.
