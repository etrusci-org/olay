Olay

Livestream overlay stuff for use as Browser-Source in OBS Studio.

1. Setup and test the overlays below. If you do not enter any values, defaults will be used.

2. When you're happy with the preview, copy & paste the URL into OBS' Browser-Source.

3. Remove all code from the CSS field of the Browser-Source. Olay will load its own default CSS for each overlay.

4. Optionally, copy & paste the default CSS (linked for each module below) into the CSS field of the Browser-Source and start adjusting it to your liking. Advanced users can also do this in the dev console of the webbrowser (navigate to the Style Editor -Tab and then style.css ) - this has the benefit of providing a realtime preview of the CSS changes.
